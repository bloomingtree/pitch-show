# 动态音符分析面板

## 功能概述

动态音符分析面板是一个集成式的设置面板，用于配置音符显示的相关参数。该面板包含两个主要功能区：

1. **风格设置** - 配置音符的视觉显示风格
2. **过滤设置** - 配置音符的过滤和合并规则

## 核心功能实现

### 1. 滚动定位高亮

使用 `IntersectionObserver` API 监听内容区域的滚动，自动高亮对应的侧边栏项。

**实现位置**：`src/components/DynamicInfoDialog.vue`

```javascript
setupScrollObserver() {
  if (!this.$refs.contentRef) return

  const options = {
    root: this.$refs.contentRef,
    threshold: 0.3
  }

  this.observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        this.activeSection = entry.target.id
      }
    })
  }, options)

  // 观察所有 section
  this.sections.forEach(section => {
    const el = document.getElementById(section.id)
    if (el) this.observer.observe(el)
  })
}
```

### 2. 配置持久化

所有配置使用 `configManager.js` 统一管理，存储在 `localStorage` 的单一 key 下。

**存储键名**：`pitch_show_settings`

**存储结构**：

```json
{
  "colorScheme": "sunset",
  "filter": {
    "minDuration": 0.05,
    "minConfidence": 0.3,
    "enableMerge": false,
    "mergeGap": 0.05
  },
  "language": "zh"
}
```

**实现位置**：`src/js/configManager.js`

### 3. 颜色方案

提供三种预设颜色方案，用于区分动态音符和稳定音符：

## 过滤设置详解

### 最短音符时长 (minDuration)

- **作用**：过滤掉持续时间过短的音符
- **默认值**：0.05 秒
- **范围**：0 - 1 秒
- **用途**：移除可能是杂音或误检测的短音符

### 最低置信度 (minConfidence)

- **作用**：过滤掉模型置信度低的音符
- **默认值**：0.3（30%）
- **范围**：0 - 1
- **用途**：移除 AI 模型不太确定的检测结果

### 自动合并短音符 (enableMerge)

- **作用**：开启后，将相邻的同音高音符合并为一个
- **默认值**：关闭
- **用途**：将碎片化的音符连接成完整的音符

### 合并间隔 (mergeGap)

- **作用**：定义两个音符之间最大允许的静音间隔
- **默认值**：0.05 秒
- **范围**：0 - 0.5 秒
- **用途**：如果两个音符之间的间隔小于此值，则合并

---

## 音符合并算法

音符合并只针对**相同音高**的音符进行，不会合并不同音高的音符。

**实现位置**：`src/js/noteMerger.js`

**算法流程**：

```
1. 按音高分组（精确匹配 pitchMidi）
   ↓
2. 每组内按开始时间排序
   ↓
3. 遍历每组，检查相邻音符：
   - 如果 gap <= mergeGap，则合并
   - 否则，保存当前音符，开始新的合并组
   ↓
4. 合并后的音符按时间排序返回
```

**合并两个音符时的处理**：

| 属性 | 合并策略 |
|------|----------|
| startTimeSeconds | 取最小值 |
| durationSeconds | 重新计算（endTime - startTime） |
| pitchMidi | 取平均值（四舍五入） |
| amplitude | 取平均值 |
| confidence | 取最大值 |
| pitchBends | 合并数组 |
| isDynamic | 根据弯音方差重新计算 |

## 动态音符判定

动态音符是指音高有明显变化的音符（如颤音、揉弦等技巧）。

**判定依据**：弯音（pitchBends）数组的方差

**计算方法**：

```javascript
// 提取弯音值
const bendValues = pitchBends.map(b => {
  if (typeof b === 'number') return b
  if (b && typeof b === 'object' && 'pitchBend' in b) return b.pitchBend
  return 0
})

// 计算方差
const avg = bendValues.reduce((a, b) => a + b, 0) / bendValues.length
const variance = bendValues.reduce((sum, val) => sum + Math.pow(val - avg, 2), 0) / bendValues.length

// 方差大于 0.05 认为是动态音符
return variance > 0.05
```
